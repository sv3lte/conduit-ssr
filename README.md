# Message Board SSR

[Message Board SSR](https://github.com/yoonge/message-board-be.git) built with Art-Template + ESlint + Joi + JWT + Koa2 + Prettier + TypeScript.

## Getting Started

```sh
$ git clone https://github.com/yoonge/message-board-be.git
$ cd message-board-be
$ pnpm i
$ pnpm run dev
```
